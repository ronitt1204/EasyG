# EasyGrocery Project for Mobile and Pervasive Computing

This project is an online grocery shopping app which will allow users to shop groceries online. One unique feature that this app will have is that each product will have its in-store aisle location so that it will be helpful to the customers who visit the store for grocery shopping and they donâ€™t have to roam around looking for stuffs they want. 

Anroid Studio is used as an IDE for this project.

Some APIs will be used to make the design better and also for a working payment gateway.
